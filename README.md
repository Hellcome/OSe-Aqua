<h1>Install</h1>

$ cd /usr/share/themes/ ; sudo git clone https://github.com/Hellcome/OSe-Aqua.git

Set this theme for elementary tweaks 

or 

$ gsettings set org.gnome.desktop.interface gtk-theme "OSe-Aqua"
